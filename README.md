# MiniJava-Compiler
Implementation of compiler for mini java
## Lexical Analysis - 
Describe the tokens of mini java by using regular expressions.

## Syntax Analysis - 
Generate Abstract Syntax Tree given the context free grammars for mini java, with LL(1) parsing.

## Semantic Analysis - 
Generate the symbol table for name and type checking, supporting method overriding, inheritance and polymorphism.

## Code Generation - 
Generate the Java assembly code by post-order traversing the Abstract Syntax Tree. Convert the Java assembly code to byte code using the assembler Jasmin.
